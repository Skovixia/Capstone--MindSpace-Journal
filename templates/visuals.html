Here is a collection of confusion matrices that were developed during training of the models used.

A confusion matrix is a table used in machine learning to visualize the performance of a classification model by presenting the counts of true positive, true negative, false positive, and false negative predictions across different classes. It helps in assessing the accuracy and effectiveness of a model's predictions.

Here is a catplot of the sentiment determined from the dataset within each emotion.

A categorical plot (catplot) using Seaborn library- It visualizes the distribution of sentiment labels across different emotions in the dataset. The x-axis represents emotions, and each emotion is further categorized by sentiment, shown by different colors. The count of each sentiment within each emotion category is represented by the height of the bars. This plot helps to understand the relationship between emotions and sentiment labels in the dataset.
